# Files used at the condefence at AHUS in May 2026

They contain 

* a directory with Hamamatsu readable data files to be visualized by NDP View2
* for Mac users:
  * a directory containing the script installing NDP View2 viewer for **_MacOS_**
  * a manual how to run this script
* for Windows users:
  * a directory containing the NDP View2 installer for **_MSWindows_**
  * a manual how to run this installer
